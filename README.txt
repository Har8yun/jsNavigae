1. START APP:

	Install node.js
	Install npm package globally npm -g install static-server
	Go to the root folder
	Run the server "npm start" or "static-server"

	OR

	Make server configs with Apache or Nginx.

	OR

	Other way you know :))

2. DESCRIPTION:

Simple "CUSTOM" vanilla js App which gets users github-info.
App get users info with input-search and shows in users-tile(table).
If users are more than "USERS_PER_PAGE" const, then pagination is working (with checking first and last pages).
Every user has his/her own user page, where is shown user info and user's repositories info.
After refresh page "remembers" it's state.
The App has simple responsive design.
Of corse sometimes you cane face some bugs! :)
